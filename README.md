# Hue-Shift

Hue Shift is an interactive web project, that changes the background color of the webpage as you move your mouse. It’s a fun and visually engaging way to explore colors.


## How to Use

1. Clone or download the repository to your local machine.
2. Open the index.html file in your web browser.
3. Move your mouse cursor on the webpage to witness the captivating hue shifts.
4. Enjoy the delightful experience of a color-changing background.
   
## Technologies Used

- HTML
- CSS
- JavaScript
